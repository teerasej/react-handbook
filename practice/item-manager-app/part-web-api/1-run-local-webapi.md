
# ดาวน์โหลด และรัน Web API

## 1. Clone Git Repo

```bash
git clone https://github.com/teerasej/note-web-api-sqlite3
```

## 2. คืนสภาพ node modules 

```bash
cd note-web-api-sqlite3
npm i
```

### ใช้ yarn ไหม?

```bash
cd note-web-api-sqlite3
yarn install
```

## 3. รัน web api

```bash
node index
```
